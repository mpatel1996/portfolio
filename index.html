Some of my past projects

Hair Segmentation

Created using python and the TensorFlow library to identify and segment human hair at a close to 80% accuracy. The model used for prediction was modified version of the default U-NET model.

Face Expression Classifier

Utilizing Python and TensorFlow library to create a program which is able to identify between three difference human facial expressions using Convolutional Neural Network and a sequential model for facial expression prediction with more than 80% accuracy.

PolyTransfer Connect

This application is developed using Flutter for android mobile devices. It utilizes Firebase as backend to store information for events, staff and resources. Using this app, users are able to stay informed about the upcoming events, on campus resources and get in touch with PolyTransfer staff member.

TheShuffling

TheShuffling is a web service for Magic The Gathering card enthusiast. Players can keep track of their physical collection virtually by creating a user profile. Website also has a global chat feature to communicate with other online users. Click here for Demo Video.

JS Typing Game

This game calls and API to randomly generate new string to type. Displays the correct and incorrect character in real time. A new random quote is generated and the timer will reset after a sentence has been typed 100% correctly.

COVID-19 Predictions

This model uses the SVM prediction tool to predict future COVID 19 cases. It also takes a user's input to predict cases for x amount of days in the future. Program compares the output between SVM and Linear Regression prediction model.
